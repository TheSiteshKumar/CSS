Understanding the RGB Color System

The RGB color model is used to represent and display images in electronic systems, such as TVs and computers. The RGB model is an additive color model in which red, green, and blue light are added together in various ways to reproduce a broad array of colors.

Basic RGB Colors

Red rgb(255, 0, 0)
Green rgb(0, 255, 0)
Blue rgb(0, 0, 255)

Combination of RGB Colors

Yellow rgb(255, 255, 0)
Cyan rgb(0, 255, 255)
Magenta rgb(255, 0, 255)
White rgb(255, 255, 255)
Black rgb(0, 0, 0)

Examples

This text is dark blue (rgb(0, 0, 128)).

Using RGB in CSS

In CSS, the RGB color model can be used to set the color of text, backgrounds, borders, and other elements. The syntax is rgb(red, green, blue), where red, green, and blue are integer values between 0 and 255, inclusive.

Here are some examples of how to use the RGB color model in CSS:


/* Set the background color to light blue */
body {
    background-color: rgb(173, 216, 230);
}

/* Set the text color to dark red */
h1 {
    color: rgb(139, 0, 0);
}

/* Set the border color to green */
div {
    border: 2px solid rgb(0, 128, 0);
}
