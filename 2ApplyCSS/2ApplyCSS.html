CSS Inclusion Methods

External CSS

External CSS is defined in an external file with a .css extension and linked to the HTML file using the <link> tag inside the <head> section.

<link rel="stylesheet" href="styles.css">

This text is styled using external CSS.

Internal CSS

Internal CSS is defined within a <style> element inside the <head> section of the HTML document.

<style>
    body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0;
        margin: 0;
        padding: 20px;
    }
</style>

This text is styled using internal CSS.

Inline CSS

Inline CSS is used to apply styles directly within an HTML element using the style attribute.

<h1 style="color: blue;">This is a heading</h1>

This text is styled using inline CSS.
