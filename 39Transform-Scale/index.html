CSS Transform: Scale Example

Hover over the box to see it scale.

Box Example

Explanation

The transform property allows you to modify the coordinate space of the CSS visual formatting model.

In this example, we use transform: scale(1.5); to scale the box to 150% of its original size when it is hovered over.

The transition property is applied to the transform property to create a smooth scaling effect.

Understanding Scale

  • The scale() function takes one or two values, where the first value scales the X-axis and the second value scales the Y-axis.
  • For example, scale(2, 1.5) would scale the width to 200% and the height to 150%.
  • Scaling can be used to create dynamic visual effects on websites.
